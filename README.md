# PowerBi_Dashboard_Project
This is PowerBi end to end project

Customer Behavior Analysis Dashboard:

Developed an interactive "Customer Behavior Analysis Dashboard" to analyze customer purchasing patterns, subscription trends, product preferences, and demographic insights. The project involved data collection, cleaning, transformation, and visualization using Excel, Python, MySQL, and Power BI.

Tools & Technologies Used:
Excel – Initial data cleaning and preprocessing
Python – Data analysis, transformation, and feature engineering
MySQL – Data storage, querying, and management
Power BI – Dashboard creation and data visualization

Key Features:

* Analyzed customer purchase amounts across different product categories.
* Examined customer distribution by subscription status and gender.
* Created age-group-based purchase and customer segmentation analysis.
* Built interactive slicers for filtering data by category, gender, subscription status, and shipping type.
* Displayed key KPIs such as Total Customers, Average Purchase Amount, and Average Review Rating.
* Developed dynamic charts including bar charts, column charts, and donut charts for business insights.
